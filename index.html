Arduino Library for the Digital Sandbox

The Digital Sandbox is a pre-wired Arduino board. It has multiple inputs and outputs that are directly connected to the microcontroller. The Digital Sandbox can be programmed using either the SparkFun DigitalSandbox board definition or using the Arduino Fio board selection in Arduino.

The Digital Sandbox (DS) has:

  • pin 13 debug LED
  • 5 LED bar graph on pins 4 - 8
  • RGB LED on pins 9, 10, and 11
  • temperature sensor
  • light sensor
  • sound sensor
  • a push button
  • slide switch
  • slide potentiometer
  • I2C auxiliary port with pins for A4 and A5
  • pin 3 expansion port with a MOSFET driver to connect motors and a buzzer directly.

To make it easier to use the Digital Sandbox, all pinModes are set with the instantiation of the digitalSandbox class. In addition, methods are added to simplify the manipulation of the LEDs and pins on the board.

To install this library, download the .zip file.

Open up Arduino. Go to: Sketch>>Include Library>>Add .ZIP Library...

Navigate to your downloads folder or where you downloaded the library, and select the file you just downloaded.
